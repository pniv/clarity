# Button Resize

<cds-alert-group status="warning" cds-layout="m-b:lg">

The `cds-action-split` component provides a action button as a split handle for
resizable type features. The component also provides two direction options `vertical` and `horizontal`.

## Installation

```typescript
import '@cds/core/button-split/register.js';
```

## Usage

```html
<cds-action-split direction="horizontal">
  <input type="range" aria-label="resize panel" value="100" min="20" max="180" step="10" />
</cds-action-split>
```

The Action Resize can be used to create resizable panels or elements.

```html
<cds-card>
  <section style="grid-template-columns: ${this.width}px 1px 1fr">
    <div class="one"></div>
    <cds-button-split>
      <input type="range" aria-label="resize panel" value="90" min="20" max="180" step="10" />
    </cds-button-split>
    <div class="two"></div>
  </section>
</cds-card>
```
